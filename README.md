### Install PHP Mailer via Github
1. Repo: https://github.com/PHPMailer/PHPMailer
2. Install as ZIP file
3. Extract and include into project folder

### Enable 2FA Gmail (Outlook, etc,..)  !!!IMPORTANT
1. Link https://myaccount.google.com/security
2. Enable 2FA 

### Add App Passwords
Link: https://myaccount.google.com/apppasswords
1. Create username
2. Save secret code

### Configure Environment variables into code
1. Change email, username, passcode
2. Attach file (optional)
